## Conceptos Clave

- **Función pura**: misma entrada → misma salida, sin efectos secundarios
- **Inmutabilidad**: los datos no se modifican, se crean nuevas versiones
- **Funciones de orden superior**: funciones que reciben o retornan otras funciones
- **Currying**: transformar una función de múltiples argumentos en una cadena de funciones de un argumento
- **Composición**: combinar funciones simples para construir comportamiento complejo
